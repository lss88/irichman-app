# NFT Club Telegram Mini App — Спецификация

## 1. Обзор проекта

**Название:** NFT Club
**Тип:** Telegram Mini App (Web App)
**Суть:** NFT-клуб с возможностью минтинга коллекции и предоставления клубных привилегий владельцам NFT
**Целевая аудитория:** Крипто-энтузиасты, коллекционеры, участники Telegram-сообщества

---

## 2. Функциональные требования

### 2.1 Авторизация и кошелёк
- Подключение кошелька через TonConnect ( поддержка Tonkeeper, TON Wallet, Safe Multisig )
- Отображение адреса кошелька и баланса TON
- Проверка владения NFT из коллекции клуба
- Привязка Telegram ID к кошельку

### 2.2 Минтинг NFT
- Mint новых NFT из коллекции клуба
- Выбор количества NFT для минта (1-10 за транзакцию)
- Отображение стоимости минта (цена в TON + комиссия)
- Отслеживание статуса транзакции
- Лимит минта на кошелёк (настраивается админом)

### 2.3 Галерея NFT
- Просмотр всех NFT коллекции клуба
- Карточка NFT: изображение, номер, владелец
- Фильтрация: все / мои NFT
- Детальная страница NFT с информацией

### 2.4 Клубные привилегии
- Раздел "Клуб" с информацией о привилегиях
- Проверка статуса владельца NFT
- Ссылки на закрытый чат (Telegram)
- Доступ к эксклюзивному контенту (для владельцев)
- История минтов пользователя

### 2.5 Админ-панель (для владельца клуба)
- Настройка цены минта
- Управление лимитами
- Просмотр статистики минтов
- Вывод средств на кошелёк админа

---

## 3. Техническая архитектура

### 3.1 Блокчейн
- **Сеть:** TON Mainnet (или тестовая сеть для разработки)
- **Стандарт NFT:** TEP-62 (Jetton Master + Collection)
- **Смарт-контракт:** Tact для NFT Collection

### 3.2 Frontend
- **Фреймворк:** React + TypeScript
- **Сборщик:** Vite
- **Telegram SDK:** @twa-dev/sdk
- **TON SDK:** @tonconnect/sdk
- **Стили:** Tailwind CSS

### 3.3 Структура проекта
```
/contracts      - Смарт-контракты Tact
/src            - React приложение
  /components   - UI компоненты
  /hooks        - React хуки (useTonWallet, useNftData)
  /pages        - Страницы (Home, Mint, Gallery, Club, Admin)
  /utils        - Утилиты
/public         - Статические файлы
```

## 5. Экраны приложения

### 5.1 Главная (Home)
- Приветствие пользователя
- Текущий статус (владелец NFT или нет)
- Кнопка быстрого минта
- Баннер с информацией о клубе

### 5.2 Минт (Mint)
- Выбор количества NFT
- Отображение стоимости
- Кнопка "Mint" с подтверждением
- История транзакций

### 5.3 Галерея (Gallery)
- Сетка NFT карточек
- Фильтры (все / мои)
- Поиск по номеру

### 5.4 Клуб (Club)
- Описание привилегий
- Список привилегий с иконками
- Кнопка входа в закрытый чат (только для владельцев)

### 5.5 Профиль (Profile)
- Адрес кошелька
- Баланс TON
- Количество NFT клуба
- Кнопка отключения кошелька

---

## 6. Критерии приёмки

1. ✅ Пользователь может подключить кошелёк через TonConnect
2. ✅ Пользователь может mint NFT за TON
3. ✅ Пользователь видит свои NFT в галерее
4. ✅ Владельцы NFT видят доступ к клубным привилегиям
5. ✅ Приложение работает внутри Telegram
6. ✅ Адаптивный дизайн для мобильных устройств
7. ✅ Обработка ошибок и состояний загрузки
